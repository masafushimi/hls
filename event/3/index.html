AI時代の Patient Centricity再設計
DTCからPSPへ繋ぐ製薬マーケティング変革

企画背景と業界変革

従来の「MR経由の有効性訴求」から、治療継続を直接支援する「Patient Support Program (PSP)」へと製薬モデルがシフトする中、患者の個別体験をどう最大化するかが勝敗を分けます。医師の働き方改革や生成AIマルチエージェントの実用化に合わせ、受診前(DTC)から受診後(PSP)をAIデータで「面」として繋ぐ新設計の最前線を探求します。

  • 患者体験価値 (PSP)への転換: 臨床外の日常を支え、治療脱落を防ぎQOLを最大化するシステム設計。
  • 生成AI・エージェントの実装: 未受診段階から治療開始後までをRWD等で一貫したジャーニーへ。
  • コンプライアンスの壁と解決策: 規制下で医師が紹介しやすい安全なAI患者フォロー体制の構築。
Seminar Concept
点から「AIによる面の設計」へ。未受診段階のDTCから治療開始後のPSPまでを生成AIとRWDで一貫したジャーニーに構築するロードマップを提示。
